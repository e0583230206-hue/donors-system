מה המערכת עושה

מערכת א בלאט גמרא היא מערכת ניהול תורמים המאפשרת לעקוב אחרי תרומות, חובות, משימות, תזכורות והודעות לחזרה. כל הנתונים נשמרים בדפדפן שלך בלבד — אין שרת, אין אינטרנט נדרש לאחר הטעינה.

כיצד לגבות את הנתונים

מומלץ לגבות לפחות פעם בשבוע. הגיבוי נשמר כקובץ JSON במחשב שלך.

1
לחץ על הגדרות בתפריט הצד.
2
לחץ על 📥 הורד גיבוי.
3
הקובץ יורד אוטומטית בשם donors-system-backup-[תאריך].json.
4
שמור את הקובץ בתיקיה בטוחה, עדיף גם בענן (Google Drive, OneDrive).

גיבוי אוטומטי

המערכת שומרת גיבוי אוטומטי בדפדפן כל 10 שמירות. לצפייה ברשימה לחץ 📋 רשימת גיבויים בדף ההגדרות. גיבוי זה נשמר בדפדפן בלבד — אינו מחליף גיבוי ידני.

מה לעשות אם הנתונים נעלמו

הנתונים נשמרים בדפדפן (localStorage). הם עלולים להימחק אם:

  • ניקית היסטוריה / cache בדפדפן
  • השתמשת במצב פרטי (Incognito)
  • הדפדפן פינה מקום אחסון אוטומטית

מה לעשות:

  1. פתח הגדרות ← גיבוי אוטומטי ← רשימת גיבויים ובדוק אם יש גיבוי אוטומטי לשחזור.
  2. אם יש גיבוי ידני בקובץ JSON — שחזר אותו לפי ההוראות למעלה.
  3. אם אין גיבוי — צור קשר עם התמיכה.
